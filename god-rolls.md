# God Roll UUID 完整列表

> 以下所有宠物均为 ★★★★★ 传说级 (Legendary) + ✦ 闪光 (Shiny)，且至少有一项属性达到 100。

## 如何使用

1. 选择你喜欢的宠物
2. 复制对应的 UUID
3. 编辑 `~/.claude.json`，将 `oauthAccount.accountUuid` 替换为该 UUID
4. 删除 `companion` 字段
5. 重启 Claude Code，输入 `/buddy hatch` 孵化

或使用一键脚本：

```bash
bash tools/quick-swap.sh <UUID>
```
