小白一键部署

复制下面的命令到终端执行，即可启动一键部署脚本。

方法一：直接运行脚本

bash tools/one-click-setup.sh

方法二：从GitHub下载并运行

curl -fsSL https://raw.githubusercontent.com/bcefghj/claude-code-buddy-guide/main/tools/one-click-setup.sh | bash

方法三：手动快速爆金（选择你喜欢的UUID复制）

手动步骤说明

  1. 备份配置: cp ~/.claude.json ~/.claude.json.bak
  2. 选择上方某个God Roll UUID
  3. 编辑配置: 编辑 ~/.claude.json
  4. 替换 oauthAccount.accountUuid 为选中的UUID
  5. 删除 companion 字段
  6. 重启 Claude Code
  7. 执行 /buddy hatch 孵化新宠物
