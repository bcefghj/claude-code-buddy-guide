1. 什么是 Buddy?

Buddy 是 Claude Code 内置的虚拟宠物系统，于 2026 年 4 月 1 日（愚人节）正式上线。它类似于电子宠物 (Tamagotchi) 和扭蛋游戏 (Gacha)。

核心特点:
  • 确定性生成: 你的宠物由账号UUID决定，同一账号永远同一只宠物
  • 18种物种: 鸭子、龙、猫头鹰、水豚等，每种都有ASCII艺术精灵
  • 5个稀有度: 普通(60%) → 传说(1%)，类似扭蛋概率
  • 闪光系统: 1%概率获得闪光版本
  • AI个性: 宠物有AI生成的名字和独特个性

2. 快速开始

前提条件

Claude Code v2.1.89 或更高版本，已登录账号。

三步上手

  1. 在 Claude Code 中输入 /buddy 查看命令菜单
  2. 输入 /buddy hatch 孵化你的宠物
  3. 输入 /buddy card 查看宠物卡片，或 /buddy pet 抚摸宠物

7. 生成算法揭秘

流程: 账号UUID + 盐值("friend-2026-401") → FNV-1a哈希 → Mulberry32 PRNG → 依次确定稀有度、物种、眼睛、帽子、闪光、属性

由于 Mulberry32 只有 32 位种子空间（约 42.9 亿个状态），用现代 CPU 暴力遍历只需几秒钟。这使得"爆金"成为可能。

// Mulberry32 伪随机数生成器
function mulberry32(seed) {
  let a = seed >>> 0;
  return function() {
    a |= 0;
    a = (a + 0x6d2b79f5) | 0;
    let t = Math.imul(a ^ (a >>> 15), 1 | a);
    t = (t + Math.imul(t ^ (t >>> 7), 61 | t)) ^ t;
    return ((t ^ (t >>> 14)) >>> 0) / 4294967296;
  };
}

如何替换宠物

  1. 备份: cp ~/.claude.json ~/.claude.json.backup
  2. 编辑 ~/.claude.json，修改 oauthAccount.accountUuid
  3. 删除 companion 字段
  4. 重启 Claude Code, 输入 /buddy hatch

或使用一键脚本: bash tools/quick-swap.sh <UUID>

11. 常见问题

Q: 修改UUID有风险吗？

目前社区反馈修改 accountUuid 不会影响功能，但建议先备份配置文件。

Q: 为什么看不到宠物？

确保 Claude Code >= v2.1.89，已输入 /buddy hatch，终端宽度足够。

Q: 传说+闪光的概率是多少？

1% × 1% = 0.01%（万分之一）。约42.9亿个种子中有约42,989,564个传说级。

Q: 宠物名字怎么生成的？

AI模型根据宠物属性和随机灵感词生成。失败时用预设名: Crumpet, Soup, Pickle, Biscuit, Moth, Gravy。
