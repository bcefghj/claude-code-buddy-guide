## 动画系统详解

### 空闲动画序列

```
帧序列: [0, 0, 0, 0, 1, 0, 0, 0, -1, 0, 0, 2, 0, 0, 0]
          ↑           ↑           ↑          ↑
        默认帧     摆动帧      眨眼帧     特殊帧

-1 = 使用帧0，但将眼睛替换为 "-" (眨眼效果)
```

### 时间轴 (15帧循环 = 7.5秒)

```
时间  0.0s  0.5s  1.0s  1.5s  2.0s  2.5s  3.0s  3.5s  4.0s  4.5s  5.0s  5.5s  6.0s  6.5s  7.0s
帧     0     0     0     0     1     0     0     0    -1     0     0     2     0     0     0
状态  静止  静止  静止  静止  摆动  静止  静止  静止  眨眼  静止  静止  特殊  静止  静止  静止
```

### 互动动画

- **抚摸** (`/buddy pet`)：触发约2.5秒的心形动画 ❤️
- **兴奋状态**：当有气泡反应或被抚摸时，快速轮播所有帧
