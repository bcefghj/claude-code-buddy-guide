┌─────────────────────────────────────────────────┐
│ 如何在 Claude Code 中使用                        │
└─────────────────────────────────────────────────┘

步骤1: 修改 ~/.claude.json

{
  "companion": {
    "name": "哆啦A梦",
    "personality": "从四次元口袋里掏出代码片段, 最讨厌老鼠(和内存泄漏)。经常用竹蜻蜓飞到代码的高层架构去查看全局。"
  }
}

步骤2: (可选) 修改 sprites.ts 源码

将 BODIES 中你的宠物物种对应的帧数据替换为上面的精灵模板。

注意:
- {E} 会被替换为你的眼睛字符
- 每帧必须是 5 行
- 尽量保持 12 字符宽度
